# Branch
{: .no_toc }

Branch commands.
{: .fs-6 .fw-300 }

## List
`bb branch list` lists branchs .

## Name
`bb branch name feature` list branches that branch name contains "feature".

## User
`bb branch user semih` list branches that latest commit user name contains "semih".
